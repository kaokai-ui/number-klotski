# Number Klotski

A Vite + React puzzle game project with three playable modes:

- `數字華容道`
- `三國華容道`
- `箱入り娘`

## Tech Stack

- React 19
- TypeScript
- Vite
- Vitest

## Development

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run tests:

```bash
npm test
```

## Notes

- The Vite dev server is configured to listen on `0.0.0.0`.
- Local planning and development notes are intentionally excluded from GitHub.
